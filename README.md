# 🎵 VibeStream

A modern, full-stack music streaming platform inspired by Spotify, built with **React**, **TypeScript**, **Vite**, **Tailwind CSS**, and **Supabase**.

VibeStream provides a beautiful, responsive interface for listening to music, creating playlists, managing favorites, and streaming songs, while giving administrators a powerful dashboard to upload and manage music.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/vibestream.git
```

Enter the project folder:

```bash
cd vibestream
```

Install dependencies:

```bash
npm install
```

Create a `.env` file and add your Supabase credentials:

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```

Run the development server:

```bash
npm run dev
```

---

# 📦 Build

```bash
npm run build
```

Preview the production build:

